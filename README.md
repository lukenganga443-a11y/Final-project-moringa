# RecipeVault

RecipeVault is a modern, lightweight recipe web application that allows users to search meals, save favorites, and store personal recipes. It is built using HTML, CSS, and JavaScript with localStorage for persistence and a clean, responsive UI.

## Features

### Recipe Search
- Search meals using a live API (TheMealDB)
- Displays recipe images and names dynamically

### Favorites System
- Add recipes to favorites
- Prevents duplicates automatically
- Stored using localStorage

### Personal Recipe Storage
- Users can add their own recipes
- Includes ingredients and cooking steps
- Saved locally in browser storage

### Persistent Data
- Favorites and custom recipes stay saved even after refresh

### Toast Notifications
- Smooth success and warning messages instead of alerts

### Responsive Design
- Works on mobile, tablet, and desktop
